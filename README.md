# Bridge-App
Мобильная аппка для обсуждения интересующих тем с единомышлиниками по всему миру

Концепция: Это не просто приложение, а мост между культурами, людьми и идеями.

Ключевая фишка: Приложение, которое в реальном времени соединяет вас с одним случайным человеком из другой страны для короткого (2-5 минут), структурированного разговора на тему, которая волнует всех.
